### 關於版本號碼設定 ###

* 請到Manifest裡面修改這兩個 Attribute
* android:versionCode="1020"
* android:versionName="1.0.2.0"
* 第一位數：更動號碼表示整個架構翻新
* 第二位數：表示部分更新
* 第三位數：代表Bug fix次數
* 第四位數：代表Build的次數
